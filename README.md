# Uber-PowerBI-Dashboard
Power BI dashboard for Uber ride-sharing analytics, tracking KPIs for bookings, revenue, and driver/rider ratings.
Project Overview

This Power BI dashboard provides a comprehensive analysis of Uber's operational and financial performance, focusing on key metrics related to bookings, revenue, distance traveled, and customer/driver ratings.

The objective of this project was to transform raw ride-sharing data into actionable business intelligence, allowing stakeholders to monitor efficiency, identify top-performing vehicle types, and pinpoint high-demand geographical locations.

Tool Used: Microsoft Power BI Desktop

Business Requirements & Analysis Goals

The dashboard was specifically designed to answer the following core business questions and provide immediate insights via Key Performance Indicators (KPIs):

Performance Monitoring: Track fundamental operational success metrics.

Trend Analysis: Analyze booking and revenue performance over time (monthly and quarterly).

Revenue Segmentation: Understand which vehicle types contribute most significantly to overall revenue.

Geospatial Demand: Identify the most frequent drop-off and pick-up locations to inform strategic resource allocation.

Quality Assessment: Monitor service quality through rider and driver ratings.

Key Performance Indicators (KPIs)

The Overview Page prominently features these immediate metrics for quick assessment:

Metric

Description

Completed Bookings

Total number of rides successfully completed.

Lost Bookings

Total number of cancelled or unfulfilled booking requests.

Total Revenue

The aggregate income generated from all completed rides.

Total Distance

The cumulative distance traveled by all vehicles.

Avg Distance

The average distance of a completed ride.

Avg Rider Ratings

Average rating given by drivers to riders.

Avg Driver Ratings

Average rating given by riders to drivers.
